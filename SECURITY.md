# Security Policy

Aura is intended to be transparent and client-side only. It should not collect telemetry, run ads, download remote code, or modify unrelated files.

## Reporting a vulnerability

Open a private security advisory on GitHub if available, or contact the maintainers privately. Please include:

- A short description of the issue
- Steps to reproduce
- Affected version or commit
- Any relevant logs or screenshots

Do not publicly disclose an exploitable issue before maintainers have had a reasonable chance to respond.
